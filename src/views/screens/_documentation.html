Documentação da API
Ver planos
GET
/travel_plans
Resposta de sucesso [{"id": 1, "travel_stops": [1,2,3]}, {"id": 2, "travel_stops": [3,2,1]}]
Ver planos expandidos ou otimizados
GET
/travel_plans?optimize=true&expand=true
Resposta de sucesso [{"id": 1, "travel_stops": [{"id": 2, name: "Earth 6", "dimension": "unknown", ...}]}, ...] optimize: Ordena as paradas de cada plano de viagem de acordo com a popularidade de cada localização de forma crescente. expand: Mostra os detalhes (nome, tipo e dimensão) de cada parada dos plano de viagens
Criar plano
POST
/travel_plans
Exemplo de requisição {"travel_stops": [1,2,3]} Resposta de sucesso {"id": 1, "travel_stops": [1,2,3]}
Ver plano
GET
/travel_plans/:id
Resposta de sucesso {"id": 1, "travel_stops": [1,2,3]}
Ver plano expandido ou otimizado
GET
/travel_plans/:id?optimize=true&expand=true
Resposta de sucesso {"id": 1, "travel_stops": [{"id": 2, name: "Earth 6", "dimension": "unknown", ...}]} optimize: Retorna as paradas (visitas) do plano de viagem :id de acordo com a popularidade de cada localização de forma crescente (do menos popular para o mais popular). expand: Mostra os detalhes (nome, tipo e dimensão) de cada parada dos plano de viagens
Editar plano
PUT
/travel_plans/:id
Exemplo de requisição {"travel_stops": [3,2,1]} Resposta de sucesso {"id": 1, "travel_stops": [3,2,1]}
Deletar plano
DELETE
/travel_plans/:id
Resposta de sucesso Status Code: 200 OK
